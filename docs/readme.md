# 🐾 Aplicativo de Rede Social para Pets e Passeios

## 📌 Sobre o Projeto

[cite_start]Este projeto é um aplicativo mobile que une o monitoramento do bem-estar animal à interação social[cite: 2, 190]. [cite_start]A plataforma foi idealizada para solucionar a falta de uma comunidade dedicada onde tutores possam compartilhar momentos e monitorar a rotina de exercícios de seus animais[cite: 190, 191]. [cite_start]O aplicativo permite o registro de passeios via GPS [cite: 97, 108][cite_start], capturando métricas [cite: 97][cite_start], e oferece um feed interativo para engajamento com outros usuários[cite: 74].

## 🎯 Público-Alvo

- [cite_start]**Tutores de Pets:** Pessoas que gostam de passear, tirar fotos, monitorar o bem-estar do seu animal e compartilhar os passeios via redes sociais[cite: 201].
- [cite_start]**Passeadores Freelancers (Dog Walkers):** Profissionais que querem ter um registro de quanto andaram com o pet e buscam uma ferramenta que ajude a gerenciar os passeios[cite: 202].

## ✨ Principais Funcionalidades

- [cite_start]**Gestão de Perfis e Pets:** Cadastro de usuários e criação, edição e exclusão de perfis de pets associados ao tutor[cite: 4, 51].
- [cite_start]**Rede Social Integrada (Feed):** Navegação por um feed de atividades [cite: 74][cite_start], publicação de mídias (fotos e vídeos) [cite: 88][cite_start], curtidas, comentários e sistema de seguidores[cite: 79, 83].
- [cite_start]**Monitoramento de Passeios (Tracking):** Rastreamento de rotas com sensor de GPS do smartphone em tempo real [cite: 99, 108][cite_start], exibindo tempo decorrido e distância percorrida[cite: 108]. [cite_start]Um passeio só pode ser iniciado se houver no mínimo um pet vinculado[cite: 120].
- [cite_start]**Co-tutoria:** Sistema que permite que o criador do perfil adicione outros donos (co-tutores) para gerenciar o animal[cite: 149]. [cite_start]Se dois co-tutores registrarem o mesmo passeio juntos, o sistema processa os dados de forma unificada para evitar duplicidade[cite: 115, 116].
- [cite_start]**Compartilhamento de Canvas:** Exportação dos dados de um passeio finalizado para outras redes sociais em formato visual (Canvas), contendo a rota, foto do pet e métricas[cite: 128, 138].

## 🛠️ Restrições e Tecnologias
