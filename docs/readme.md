# 🐾 Aplicativo de Rede Social para Pets e Passeios

## 📌 Sobre o Projeto

Este projeto é um aplicativo mobile que une o monitoramento do bem-estar animal à interação social. A plataforma foi idealizada para solucionar a falta de uma comunidade dedicada onde tutores possam compartilhar momentos e monitorar a rotina de exercícios de seus animais. O aplicativo permite o registro de passeios via GPS , capturando métricas, e oferece um feed interativo para engajamento com outros usuários.

## 🎯 Público-Alvo

- **Tutores de Pets:** Pessoas que gostam de passear, tirar fotos, monitorar o bem-estar do seu animal e compartilhar os passeios via redes sociais.
- **Passeadores Freelancers (Dog Walkers):** Profissionais que querem ter um registro de quanto andaram com o pet e buscam uma ferramenta que ajude a gerenciar os passeios.

## ✨ Principais Funcionalidades

- **Gestão de Perfis e Pets:** Cadastro de usuários e criação, edição e exclusão de perfis de pets associados ao tutor.
- **Rede Social Integrada (Feed):** Navegação por um feed de atividades, publicação de mídias (fotos e vídeos) , curtidas, comentários e sistema de seguidores.
- **Monitoramento de Passeios (Tracking):** Rastreamento de rotas com sensor de GPS do smartphone em tempo real, exibindo tempo decorrido e distância percorrida. Um passeio só pode ser iniciado se houver no mínimo um pet vinculado.
- **Co-tutoria:** Sistema que permite que o criador do perfil adicione outros donos (co-tutores) para gerenciar o animal. Se dois co-tutores registrarem o mesmo passeio juntos, o sistema processa os dados de forma unificada para evitar duplicidade.
- **Compartilhamento de Canvas:** Exportação dos dados de um passeio finalizado para outras redes sociais em formato visual (Canvas), contendo a rota, foto do pet e métricas.

## 🛠️ Restrições e Tecnologias

O escopo foca em uma plataforma mobile, utilizando integrações com GPS e adequação à LGPD.
